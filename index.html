Our Visualization

Writeup So Far

So far, we have created this visualization which uses 2015 as a baseline and shows how weather has changed over time. The visualization allows the user look into weather changes till 2025, by showing blue or red spots indicating a decrease or increase respectively. The visualization works by having a slide bar for the user to change the year that is currently being shown, the user can also see the exact change in degrees (celsius) in specific areas. The visualization mainly focuses on North America. We forsee the challenging parts to be the expansion of our visualization, in area and time. Area wise, it will be challenging to fit all the data of the world into this visualization as it kind of breaks our code and makes it wonky. Time is even more complicated, because we would like to implement a feature where it shows potential future weather changes after 2025. This will be challenging, as we have to figure out how to use the data to predict the future changes in weather for specific areas.

Dataset: Coupled Model Intercomparison Project (CMIP6)

Our Visualization

Design Decisions and Rationale:

  • Using 2015 as the baseline year. We compare each year to 2015 so the map emphasizes temperature change over time rather than raw temperature. This makes the visualization easier to interpret because viewers can focus on warming or cooling patterns instead of comparing absolute temperatures across different regions.
  • Using a red-blue color scale for anomalies. Red is used to represent warmer-than-baseline areas, while blue represents cooler-than-baseline areas. This color choice is intuitive for temperature data and helps users quickly recognize where the strongest warming appears.
  • Adding interactivity through a year slider and tooltip. The slider lets users move through time and observe spatial changes year by year. The tooltip provides specific coordinates and anomaly values, giving both a broad global overview and detailed local information.

Final Reflection

Building this visualization showed us how much visual storytelling depends on framing. Even when using climate model data, the choices we make about baseline year, color scale, projection, and interaction design all shape how the viewer understands the pattern. By using 2015 as a baseline, the map focuses attention on change over time rather than raw temperature differences between naturally hot and cold regions.

The most important ethical design choice was making the comparison clear. A temperature anomaly map can be misleading if viewers do not understand what the colors are being compared against. That is why the title, legend, tooltip, and caption all state that the values represent change since 2015. This helps prevent readers from interpreting the colors as absolute temperatures.

Overall, this project helped us see that an effective visualization should be both persuasive and transparent. The map is designed to make warming patterns visually noticeable, but it also gives users enough context to understand what the data actually represents.
